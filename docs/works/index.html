Aplicaciones Dinámicas

En Aplicaciones Dinámicas me encargué de realizar mantenimiento y actualización de sistemas existentes haciendo uso de Symfony, el cual se actualizó de la versión 4 a la versión 6.3. En este proceso se usaron herramientas de AWS como S3, lambda, RDS con MySQL. ElasticBeanStack para realizar el despliegue de los sistemas. Además, estuve encargado de desarrollar las API requeridas por el sistema usando arquitectura de microservicios en este proceso. Estuve a cargo de migrar las páginas de documentación y las páginas estáticas haciendo uso de Hugo Framework. También implementé scrape haciendo uso de Python con el objetivo de generar campañas de marketing
